    RGToastSample

    RGToastSample is a sample app to show how RGToast works. It doesn't do anything other than that.

    How To Run

    You will need CocoaPods to run RGToastSample.

    Run pod install from the root RGToastSample directory. Then open the (newly created) GToastSample.xcworkspace file.
